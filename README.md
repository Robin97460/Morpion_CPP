# Projet : Tic Tac Toe (C/C++)

Ce dépôt contient le code source d'un jeu de Tic Tac Toe (Morpion) développé en C/C++ pour le terminal. Ce projet a été réalisé en groupe dans le cadre d'une évaluation technique mettant l'accent sur le développement C++ et les bonnes pratiques Git/GitHub.

## Description du projet

Le but est de proposer un jeu de stratégie classique pour deux joueurs sur une grille de 3x3. [cite_start]Le projet respecte une architecture modulaire avec une séparation claire du code (fichiers `.h` et `.cpp`).

### Fonctionnalités principales 
 **Grille de jeu** : Plateau classique de 3x3 cases.
 **Mode 2 Joueurs** : Tour par tour pour les joueurs 'X' et 'O'.
 **Validation des saisies** : Gestion des erreurs si un joueur entre une case invalide ou déjà occupée.
 **Logique de jeu** : Détection automatique de la victoire (lignes, colonnes, diagonales) ou de l'égalité (match nul).
 **Interface** : Affichage clair du résultat final dans le terminal.

### Bonus implémentés 

**Mode IA** : Possibilité de jouer contre l'ordinateur.
